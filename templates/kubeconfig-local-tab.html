Download and Install the OIDC Auth Plugin

You must download the kubelogin plugin to authenticate with kubectl using OIDC.

  1. Visit kubelogin releases page
  2. Important: Select the correct binary for your operating system and CPU architecture:
    • Linux: Choose `kubelogin_linux_amd64`
    • Windows: Choose `kubelogin_windows_amd64.exe`
    • macOS: Choose `kubelogin_darwin_amd64` (Intel) or `kubelogin_darwin_arm64` (Apple Silicon)
  3. Uncompress the file
  4. Rename the file: `mv kubelogin kubectl-oidc_login` (macOS/Linux only)
  5. Make it executable: `chmod +x kubectl-oidc_login` (macOS/Linux only)
  6. Place it in your PATH (e.g., `/usr/local/bin` on macOS/Linux or add to PATH on Windows)

Testing

To test that the plugin is installed correctly, run:

$ kubectl oidc-login
No configuration found.
You need to set up the OIDC provider, role binding, Kubernetes API server and kubeconfig.
To show the setup instructions:

    kubectl oidc-login setup

If this message is displayed, it means the plugin was successfully installed.

Put this into ~/.kube/config - IF YOU ARE A LINUX OR macOS USER

For Windows users click here

apiVersion: v1
clusters:
- cluster:
    certificate-authority-data: {{ b64enc .K8sCaPem }}
    server: {{ .K8sMasterURI }}
  name: {{ $CLUSTER_NAME }}
contexts:
- context:
    cluster: {{ $CLUSTER_NAME }}
    user: {{ .Username }}-{{ $CLUSTER_NAME }}
  name: {{ .Username }}-{{ $CLUSTER_NAME }}
current-context: {{ .Username }}-{{ $CLUSTER_NAME }}
kind: Config
preferences: {}
users:
- name: {{ .Username }}-{{ $CLUSTER_NAME }}
  user:
    exec:
      apiVersion: client.authentication.k8s.io/v1
      interactiveMode: IfAvailable
      provideClusterInfo: false
      command: kubectl
      args:
        - oidc-login
        - get-token
        - --oidc-issuer-url={{ .Issuer }}
        - --oidc-client-id={{ .ClientID }}
        - --oidc-client-secret={{ .ClientSecret }}
        - --oidc-extra-scope=email
        - --oidc-extra-scope=profile
        - --oidc-extra-scope=groups
        - --token-cache-dir=~/.kube/cache/login/{{ $CLUSTER_NAME }}
